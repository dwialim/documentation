    Buat dashboard pertama Anda¶

    Topik ini membantu Anda memulai Grafana dan membangun dashboard pertama Anda menggunakan sumber data bawaan Grafana. Untuk mempelajari lebih lanjut tentang Grafana, silakan lihat Pengantar Grafana.

    CATATAN

    Grafana juga menawarkan akun gratis dengan Grafana Cloud untuk membantu Anda memulai dengan lebih mudah dan cepat. Anda dapat menginstal Grafana untuk hosting mandiri atau mendapatkan akun Grafana Cloud gratis

    Install Grafana¶

    Grafana dapat diinstal di berbagai sistem operasi. Untuk daftar persyaratan perangkat keras dan perangkat lunak minimum, serta petunjuk penginstalan Grafana, silakan lihat bagian Install Grafana.

    Sign in to Grafana¶

    Untuk masuk ke Grafana untuk pertama kalinya:

    1. Buka web browser Anda dan kunjungi http://localhost:3000/.
      Port HTTP default yang digunakan Grafana adalah 3000, kecuali Anda telah mengonfigurasi port yang berbeda.

    2. Pada halaman masuk, masukkan admin untuk username dan password.

    3. Klik Sign in.
      Jika berhasil, Anda akan melihat permintaan untuk mengubah password.

    CATATAN

    Kami sangat menyarankan Anda mengubah password administrator default.

    Create a dashboard¶

    Jika Anda sudah menyiapkan sumber data yang Anda ketahui cara melakukan kuerinya, lihat Create dashboard.

    Untuk membuat dashboard pertama Anda menggunakan sumber data bawaan -- Grafana --:

    1. Klik Dashboard di menu utama
    2. Pada halaman Dashboard, klik New dan pilih New Dashboard dari drop-down menu.
    3. Pada Dashboard, klik + Add visualization.
    4. Pada kotak dialog yang terbuka, klik -- Grafana --: Ini mengonfigurasikan query Anda dan menghasilkan Random Walk dashboard.
    5. Klik Refresh untuk query data source.
    6. Setelah selesai mengedit panel, klik Save dashboard.
      Atau, klik Back to dashboard jika Anda ingin melihat perubahan diterapkan ke dashboard terlebih dahulu. Kemudian, klik Save dashboard jika sudah siap.
    7. Tambahkan judul deskriptif untuk dashboard, atau minta Grafana membuatnya menggunakan fitur generatif AI, lalu klik Save.
    8. Klik Back to dashboard dan kemudian Exit edit.

    Selamat, Anda telah membuat dashboard pertama Anda dan menampilkan hasilnya.
